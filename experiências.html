Ford

11/2021 - atualmente

Atuo como bolsista na Ford, participando do projeto que visa construir uma plataforma de cálculos de junta, durante esta oportunidade sempre me dedico aprimorar e aprender mais todos os dias através de reuniões, tanto com outras equipes de bolsistas quanto com meus supervisores, além de explorar as plataformas de estudo disponibilizadas pela Ford.

Utilizo Python para fazer cálculos complexos que vão desde manipulações de matrizes até características únicas das juntas, transformando tudo isso em uma plataforma com HTML, CSS3, Bootstrap, Django e armazenando os dados das juntas, usuários e materiais da Ford através do PostgreSQL.

Na Ford estou atuando nas seguintes responsabilidades: utilizando PostgreSQL para construção do banco de dados, utilizando e aprendendo mais sobre JavaScript, criando Back-end com Python e Django framework, realizando a analise dos cálculos de juntas feitos pela empresa, utilizando as ferramentas de Front-end: HTML5, Bootstrap e CSS3.
